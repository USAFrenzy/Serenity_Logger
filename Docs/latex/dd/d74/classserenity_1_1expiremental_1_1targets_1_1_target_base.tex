\doxysubsection{Detailed Description}
The base class that derived target classes inherit from. Contains common functions between all targets for message logging and logging settings. 

For All \doxyref{Target\+Base}{p.}{dd/d74/classserenity_1_1expiremental_1_1targets_1_1_target_base} Constructors\+: ~\newline
The following default values are set regardless of which constructor is used\+: ~\newline

\begin{DoxyItemize}
\item Flush Policy is set to \char`\"{}never\char`\"{}. ~\newline

\item Log Level is set to \char`\"{}\+Logger\+Level\+::trace\char`\"{}. ~\newline

\item Default pattern is set to \char`\"{}$\vert$\%l$\vert$ \%x \%n \%\+T [\%\+N]\+: \%+\char`\"{} which translates to the form of\+: ~\newline
 \char`\"{}$\vert$\+T$\vert$ Sat 29\+Jan22 [\+Base\+\_\+\+Logger]\+: The message to log\char`\"{}. ~\newline

\item Time mode used is set to \char`\"{}time\+\_\+mode\+::local\char`\"{}.~\newline
 
\end{DoxyItemize}

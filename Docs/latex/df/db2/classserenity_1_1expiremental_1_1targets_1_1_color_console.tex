\doxysubsection{Detailed Description}
This class is in charge of logging to the terminal and supports color logging and the ability to log without color. This class inherits from the \doxyref{Target\+Base}{p.}{dd/d74/classserenity_1_1expiremental_1_1targets_1_1_target_base} class for common logging functions and logging settings. 

For all Console Target Constructors\+: ~\newline

\begin{DoxyItemize}
\item will set the console mode to \char`\"{}console\+\_\+interface\+::stdout\char`\"{} ~\newline

\item Initializes the default colors to use for each message level ~\newline

\item If output is a terminal and hasn\textquotesingle{}t been redirected and if the output handle is valid, enables color output ~\newline

\item Sets \char`\"{}\+ENABLE\+\_\+\+VIRTUAL\+\_\+\+TERMINAL\+\_\+\+PROCESSING\char`\"{} flag if on Windows Platform ~\newline
 
\end{DoxyItemize}
